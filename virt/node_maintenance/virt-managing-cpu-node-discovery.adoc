= Managing CPU node discovery
include::modules/virt-document-attributes.adoc[]
include::modules/common-attributes.adoc[]
:context: virt-managing-cpu-node-discovery
toc::[]

The CPU node discovery feature enables you to schedule virtual machines (VMs) on nodes that support specific CPU models and policy attributes. A VM's CPU model and hardware features are matched for compatibility with the CPU model and policy attribute supported by the node. Then, the VM is scheduled on that node.
